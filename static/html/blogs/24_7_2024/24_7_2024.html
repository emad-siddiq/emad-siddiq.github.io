The C Tutorial Part 2.

Hello and welcome back! So without any further ado, we will quickly recap that last time we learnt about how to write C functions and run them on our machines.

Recall the while loop from the previous tutorial. It helps us repeatedly run a block of code that it encapsulates inside curly braces {}.

Today we are going to introduce a very similar concept called the for loop. The main difference is that instead of just checking for a condition, the () include three statements terminated by a semicolon ;. The first initializes variables, the second checks the condition, the third updates the variables. In this way we end up with a more nuanced means of repeatedly running a block of code.

For example, the previous while loop Fahrenheit-Celsius convertor can be rewritten as: 

--- f_to_c.c ---
#include <stdio.h> 

int main() {
    int fahr;
    for (fahr = 0; fahr <= 300; fahr = fahr + 20) {
        printf("%4d %6.1f\n", fahr, (5.0/9.0) * (fahr-32));
    }
}
We first declare a variable fahr to be of type int. Then in the 3 step loop condition:
  1. We first initialize fahr to be 0. We then run the body of the for loop and print fahr as well as
    the calculated value of converting the current fahr value to Celsius.
  2. After it has run once, we check if fahr is less than 300 with fahr <= 300;
  3. If it is, we add 20 to fahr. Otherwise the for loop ends.

Exercise 1-5

The goal of Exercise 1-5 is to edit the code above to have it print the table in reverse order i.e. from 300 to 0. This is not an unreasonable request and we can implement it with a few changes: 

--- f_to_c.c ---
#include <stdio.h> 

int main() {
    int fahr;
    for (fahr = 300; fahr >= 0; fahr = fahr - 20) {
        printf("%4d %6.1f\n", fahr, (5.0/9.0) * (fahr-32));
    }
}
We initialize fahr at 300 and subtract 20 while fahr is greater than or equal to 0.

We can improve on this code with symbolic constants to store constants.
During compilation, unquoted occurences of the symbolic constant are replaced with the corresponding string. 

--- f_to_c.c ---
#include <stdio.h> 

#define LOWER 0
#define UPPER 300
#define STEP 20

int main() {
    int fahr;
    for (fahr = UPPER; fahr >= LOWER; fahr = fahr - STEP) {
        printf("%4d %6.1f\n", fahr, (5.0/9.0) * (fahr-32));
    }
}
